# RDNS
Reverse DNS Program. Takes IPv4 addresses and converts them into their respective domain names.

# Usage
./rdns 8.8.8.8

Takes an IPv4 address as a parameter and returns the respective domain name.

# Building
## Requirements:
- GNU C Compiler

## Dependencies
- Standard POSIX environment

## Process
Build process
```
make all
```
